# CareerGPS – Premium EdTech Redesign System

## 🎨 UI Spacing System (Professional Grid Layout)

- **Base Grid:** 8px spacing system (multiples: 8, 16, 24, 32, 40, 48, 64, 96, 128)
- **Card Padding:** 24px (desktop), 16px (mobile)
- **Section Padding:** 64px top/bottom (desktop), 32px (mobile)
- **Card Gap:** 24px (desktop), 16px (mobile)
- **Navbar Height:** 64px (desktop), 56px (mobile)
- **Button Padding:** 16px 32px (primary), 12px 24px (secondary)
- **Grid Columns:**
  - Desktop: 12-column grid, 80px max content width, 24px gutter
  - Tablet: 8-column grid, 16px gutter
  - Mobile: 4-column grid, 8px gutter

## 🧠 UX Psychology Reasoning (Color)

- **White:** Trust, clarity, focus, and approachability
- **Blue (#2563EB):** Reliability, calm, academic authority
- **Purple (#7C3AED):** Creativity, inspiration, growth
- **Green (#22C55E):** Success, progress, positive reinforcement
- **Amber (#F59E0B):** Motivation, warning, attention
- **Gradients:** Signal progress, energy, and modernity
- **No red/black:** Avoids stress, feels safe and welcoming
